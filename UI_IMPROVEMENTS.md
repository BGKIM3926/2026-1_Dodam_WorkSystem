# WorkHistoryForm UI 개선 사항

## 개요
WorkHistoryForm 컴포넌트의 UI를 현대적이고 전문적인 스타일로 개선했습니다. 모든 기존 기능을 유지하면서 시각적 품질을 향상시켰습니다.

## 디자인 철학
`Swiss Clean` 디자인 원칙을 기반으로 개선:
- **최소주의**: 불필요한 요소 제거
- **명확한 계층 구조**: 타이틀, 부제목, 폼 필드, 버튼의 명확한 계층
- **깔끔한 경계**: 실선 테두리를 사용한 깔끔한 스타일
- **일관된 간격**: 통일된 24px 필드 간격

## 주요 개선 사항

### 1. 타이포그래피 개선
```
- 제목: 32px, font-weight 600, #18181b
- 부제목: 14px, font-weight 400, #71717a (설명 추가)
- 필드 레이블: 14px, font-weight 600, #18181b
- 입력 필드: 15px, font-weight 400
```

### 2. 색상 시스템
```
Primary Text: #18181b (칙칙한 검은색)
Secondary Text: #71717a (회색)
Tertiary Text: #a1a1aa (밝은 회색)
Border: #e4e4e7 (1.5px 스트로크)
Accent: #2563eb (파란색 - 버튼 및 포커스)
```

### 3. 형태 (Border Radius)
```
- 입력 필드: 8px (약간 둥근 모서리)
- 버튼: 8px
```

### 4. 레이아웃 개선
```
이전:
- Grid 기반 레이아웃
- 레이블 없는 입력 필드
- 일관되지 않은 간격

현재:
- Flexbox 기반 깔끔한 레이아웃
- 명확한 레이블
- 일관된 24px 필드 간격
- 32px 섹션 간격
```

### 5. 인터랙션 개선

#### 입력 필드 상태
- **기본 상태**: #e4e4e7 테두리
- **호버**: #d4d4d8 테두리 (약간 진해짐)
- **포커스**: #2563eb 테두리 + 파란색 배경 음영 (0.1 opacity)

#### 버튼
**뒤로가기 버튼**:
- 기본: 흰색 배경 + #e4e4e7 테두리
- 호버: #fafafa 배경 + #d4d4d8 테두리

**등록 버튼**:
- 기본: #2563eb 배경 + 흰색 텍스트
- 호버: #1d4ed8 (진한 파란색)
- 활성: #1e40af (매우 진한 파란색)

### 6. 입력 필드 표준화
```
- 높이: 44px (통일된 높이)
- 패딩: 12px (좌우)
- 테두리: 1.5px solid
- 폰트: 15px Inter
```

### 7. 텍스트 에어리어 개선
```
- 최소 6줄 표시 (minRows={6})
- 줄 길이: 1.5 (읽기 편함)
- 자동 줄바꿈: pre-wrap (형식 유지)
- 단어 줄바꿈: break-word (자연스러운 줄바꿈)
```

## 기능 유지
모든 기존 기능이 100% 유지됩니다:
- ✅ Form 상태 관리
- ✅ 시스템 선택 드롭다운
- ✅ 작업 유형 선택
- ✅ 장비 입력
- ✅ 내용 입력 (자동 줄바꿈)
- ✅ 뒤로가기 네비게이션
- ✅ 등록 제출 기능

- Material-UI v5
- React 18+
- sx prop (styled-components 대체)
